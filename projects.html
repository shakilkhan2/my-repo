How to creat a new repository to GitHub

First of all we have to know what is GitHub? GitHub is a host site where we can put our code and other things. To put our code to GitHub, we have to creat repository to GitHub. So now we will see how to creat a repository to GitHub.

There are few steps to creat a new GitHub Repository:

  1. At first we have to creat a GitHub account.
  2. Click the new repository
  3. Write a name "my-first-repo"
  4. Then creat a new repository
  5. Now creat a new folder to your computer named 'that you named to git'
  6. Open it in the vs code

    7. Go to terminal and follow these steps:
    • git init [press enter]
    • git add . [careful about space & spelling then enter]
    • git commit -m"write here a meaningful messege" [enter]
    • git branch -M main [enter]
